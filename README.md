# RTX (rtx)
RTX Corporation, formerly Raytheon Technologies, is an American multinational aerospace and defense conglomerate headquartered in Arlington, Virginia. RTX provides advanced defense systems and technologies, including the EAGLE platform which offers API integration capabilities for intelligence analysis and data management in government environments.

## APIs

### EAGLE API
The RTX EAGLE API is part of the EAGLE application bundle developed by Raytheon. EAGLE provides intelligence analysis and data management capabilities, with the API allowing integration with EAGLE Vision and EAGLE Web Portal for analytics and intelligence workflows.
