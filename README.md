# RTX (rtx)

RTX Corporation is a leading American aerospace and defense company comprising three market businesses: Collins Aerospace, Pratt & Whitney, and Raytheon. Raytheon develops the EAGLE (Enhanced Automated Graphical Logistics Environment) software platform for integrated logistics support and logistic support analysis across defense programs. RTX BBN Technologies (a Raytheon subsidiary) develops open-source software including SPARQL triple stores, NLP frameworks, and TAK ecosystem plugins for government and military situational awareness platforms.

## APIs

### RTX EAGLE API

The RTX EAGLE (Enhanced Automated Graphical Logistics Environment) API provides integration capabilities for the EAGLE logistics support analysis platform developed by Raytheon. EAGLE manages product support throughout a defense system's entire lifespan, enabling users to build and maintain logistics data, create reports, and support fielded systems through operations and maintenance. The API supports integration with intelligence analysis, data management, and analytics workflows used in defense and government environments.
